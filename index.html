Prework Study Guide

HTML

  • The head element contains information about the webpage
  • the body element represents visible content the user can see
  • the p(aragraph) element represents a block of text in a paragraph
  • the h(1 through 6) element represents headers that are largest to smallest from 1 to 6
  • the img element is used to place an image file and requires a src attribute describing where the image is stored and an alt attribute that shows alternative text for when the image does not load
  • footer element is just to creater a footer at the bottom of the page

GIT

  • git status: this checks what branch currently in
  • git checkout -b 'branch-name' creates a new branch and switches to it

  • git pull origin main: sync local env with main branch of github repo
  • git push origin 'branch name': send local changes to github repository
